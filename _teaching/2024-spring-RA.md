This is an undergraduate course in Peking University, instructed by [Prof. Yuqing Kong](https://cfcs.pku.edu.cn/yuqkong/). This course introduces several randomized algorithms and mathematical tools for analyzing them.

Lectures
======
Monday, 5th to 6th sessions (13:00-14:50), Room 306, No.2 Teaching Building

Thursday, 3th to 4th sessions (10:10-12:00), Room 306, No.2 Teaching Building


Contents
======
1. Randomized algorithms
  - Identity testing
  - Perfect matching
  - Primality testing
2. Probabilistic analysis
  - Probabilistic method
  - Second moment method
  - Psuedo-randomness and universal hashing
  - Unbiased estimator
  - Chernoff bound
  - Stochastic dominance
  - Dimensional reduction
  - Lovasz’s Local Lemma
3. Stochastic process
  - Branching Process
  - Markov Chain
  - Martingale and Azuma’s Inequality
  - Optional Stopping Theorem
4. A lot of examples in application
